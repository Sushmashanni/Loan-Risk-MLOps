# Loan Risk Prediction using MLOps

## Project Overview

This project implements an end-to-end MLOps pipeline for predicting loan risk using machine learning.

## Technologies Used

- Python
- Scikit-Learn
- DVC
- MLflow
- FastAPI
- Docker
- GitHub Actions
- Prometheus
- Evidently AI

## Project Structure

- Data Versioning using DVC
- Experiment Tracking using MLflow
- FastAPI REST API
- Docker Containerization
- CI/CD using GitHub Actions
- Monitoring using Prometheus
